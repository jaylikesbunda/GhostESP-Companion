# GhostESP Companion

Android companion app for GhostESP devices.

## Requirements

- Android 7.0+ (API 24)
- Android Studio
- JDK 17

## Build

```bash
./gradlew assembleDebug
```

## Features

- USB serial communication with GhostESP devices
- WiFi scanning and management
- Bluetooth device interaction
- Evil Portal functionality
- NFC operations
